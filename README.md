# Real-Time Vehicle Detection and Traffic Analysis

## Overview
This project uses OpenCV and Python to detect, track, and count vehicles from traffic video footage. The system applies frame differencing, contour detection, and centroid tracking techniques to monitor traffic flow in real time.

## Features
- Vehicle detection
- Vehicle tracking
- Vehicle counting
- Traffic monitoring
- Computer vision-based analytics

## Technologies Used
- Python
- OpenCV
- NumPy
- Jupyter Notebook

## Project Structure
- vehicle detection.ipynb : Main implementation notebook
- README.md : Project documentation
